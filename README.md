# DeleteHub

Official Account Deletion Links – Quick access to permanent account deletion pages for 25+ major platforms.

DeleteHub is a simple, fast directory that helps users directly navigate to official account deletion pages without searching through confusing help centers.


## Live Website

https://deletehub.vercel.app/

## Purpose

Many platforms make account deletion difficult to find.

DeleteHub solves this by:

- Providing direct official deletion links
- Organizing apps in a clean, searchable grid
- Maintaining a lightweight, fast-loading interface

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Deployed on Vercel
- SEO optimized (robots.txt + sitemap.xml)

No backend. No database. Fully static and optimized.
